What is this

This is a game of chutes and ladders with one change: The chutes and ladders will send you to random locations instead of fixed ones.

The tiles with green borders are ladders. If you land on a ladder it will send you to a random tile ahead.

The tiles with the red borders are chutes. If you land on a chute, it will send to a random tile behind.

How to play

Select the number of players below and click start.

Click on the ‘Roll P’ button to roll the dice and start.

Click the same button again to roll for the next player.
